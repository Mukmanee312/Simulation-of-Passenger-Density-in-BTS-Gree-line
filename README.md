# 🚆 Simulation of Passenger Density in BTS Green Line

โปรเจกต์นี้เป็นการจำลองความหนาแน่นของผู้โดยสารในรถไฟฟ้า BTS สายสีเขียว  
เพื่อวิเคราะห์รูปแบบการเดินทางและช่วยในการวางแผนระบบขนส่ง จำลองการแก้ปัญหาผู้โดยสารหนาแน่น

---

## 📌 Features
- วิเคราะห์จำนวนผู้โดยสารในแต่ละสถานี
- จำลองความหนาแน่นในช่วงเวลาเร่งด่วน 
- ใช้ข้อมูลเพื่อช่วยตัดสินใจด้านการจัดการระบบ
